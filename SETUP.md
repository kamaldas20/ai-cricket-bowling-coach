# Setup Instructions

## Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install the necessary packages:
   ```bash
   npm install
   ```

3. Configure the port (if applicable):
   - Default port is **3000**. To change, modify the `.env` file.

## Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install the necessary packages:
   ```bash
   npm install
   ```

3. Configure the port (if applicable):
   - Default port is **5000**. To change, modify the server configuration file.

## AI Engine Setup

1. Navigate to the AI engine directory:
   ```bash
   cd ai_engine
   ```

2. Install the necessary packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure the port (if applicable):
   - Default port is **8000**. To change, modify the configuration settings in the AI engine.
